# Task

Implement two scripts:

- `src/training/train.py`
- `src/training/test.py`

These scripts will handle model training and prediction.

---

# 1. train.py Requirements

This script should:

1. Load:
   - `train.csv`
   - `val.csv`
   - `target.csv`

2. Merge features with targets using the index:


['ggo', 'odsek', 'leto_mesec']


3. Train a predictive model.

4. Handle **strong class imbalance**, since:


~90% of target values are 0


Use appropriate techniques such as:

- class weighting
- resampling
- imbalance-aware models

5. Evaluate model performance on the validation set.

6. Save the trained model to:


models/model.pkl


---

# 2. test.py Requirements

This script should:

1. Load:

- `test.csv`
- `target.csv` (if needed)
- `models/model.pkl`

2. Generate predictions for the test set.

3. Save predictions to:


predictions/test_predictions.csv


The output file must:

- Preserve the original index:


['ggo', 'odsek', 'leto_mesec']


- Contain one 12 column:

[h1, h2, ..., h12]

- Choose evaluation metrics appropriate for imbalanced classification.
